  • Online register and order form

    These forms are built by me without any help from my coworker. They allow the customers to register their information and order products online. It is consisted if HTML, CSS, and PHP. Please do not fill out the forms and submit them since they might lead to sending a request of buying the products if the website is still being used

  • Auto Marking App (online & Desktop)

    I build the prototype of the app which allows student to do quizzes online. The online app will then auto mark the multiple choice questions and send the questions and answers to teacher to mark the written questions. Teachers can also login to the web page to mark the questions online. This app is consisted of HTML, CSS, and PHP. The desktop version, on the other hand, is consisted of C++. It is a Console app that reads students answer and produce the mark written in separate text file. This prototype later on is used in a presentation and help the company to gained a contract from a university.



Ultimate Software Group, Inc. - Software Test Engineering


  • API Unit Testing

    As a tester, my primary gaol in this coop term is to maintain the already existing unit tests and create new tests for the new features of the products. These are done by C# using Microsoft Visual Studio. In addition, the company decide to make up some API testing using Postman and Visual Studio. They brought in two co-op students to start the project and I was one of them. The test was about sending JSON through web API and get back a JSON to see of the functionalities if the web products is working or not. As a result, we had to consist the JSON using C# and sent it through the web API then verified the returned JSON is the expected one. These all have to be automated as unit tests. The project was a success and nearly 200 tests were created for the first try and they ran very fast.
